# Branche de similarité

La branche de similarité regroupe les cours, concepts et compétences qui reviennent dans plusieurs certificats.

Son objectif est de montrer les fondations communes que je dois vraiment maîtriser.

## Rôle de cette branche

Cette branche répond à la question :

> Qu’est-ce que mes certificats ont en commun ?

Au lieu de répéter le même cours plusieurs fois, Engineer Method OS transforme les cours similaires en compétences centrales.

## Fondations communes

### 1. Software Engineering

Compétence :
- comprendre le cycle de développement logiciel ;
- analyser un problème ;
- concevoir une solution ;
- tester et améliorer une application.

Explication simple :
Le software engineering est la méthode professionnelle pour créer un logiciel de manière organisée.

---

### 2. Git et GitHub

Compétence :
- sauvegarder le code ;
- suivre les changements ;
- travailler avec des branches ;
- montrer son travail aux employeurs.

Explication simple :
Git permet de suivre l’évolution d’un projet. GitHub permet de partager ce projet en ligne.

---

### 3. HTML, CSS et JavaScript

Compétence :
- construire une page web ;
- organiser le contenu ;
- styliser l’interface ;
- ajouter de l’interactivité.

Explication simple :
HTML construit la structure, CSS donne le style, JavaScript rend la page vivante.

---

### 4. React

Compétence :
- créer une interface moderne ;
- organiser l’application en composants ;
- afficher des données ;
- créer un tableau de bord.

Explication simple :
React permet de construire l’interface visible de l’application.

---

### 5. Node.js et Express

Compétence :
- créer un serveur ;
- créer une API ;
- recevoir des demandes ;
- envoyer des réponses.

Explication simple :
Node.js et Express permettent de construire le backend d’une application.

---

### 6. Python et Flask

Compétence :
- créer des applications Python ;
- construire des API simples ;
- connecter l’IA à une application ;
- analyser des données.

Explication simple :
Python est utile pour l’IA, les données et les outils intelligents. Flask permet de créer une application web légère avec Python.

---

### 7. Docker, Kubernetes et OpenShift

Compétence :
- préparer une application pour le déploiement ;
- exécuter l’application dans des conteneurs ;
- gérer une application comme un professionnel.

Explication simple :
Ces outils permettent de rendre une application plus portable, stable et professionnelle.

---

### 8. Microservices et Serverless

Compétence :
- diviser une grande application en petits services ;
- construire une architecture moderne ;
- rendre une application plus flexible.

Explication simple :
Au lieu de construire une seule grosse application, on peut créer plusieurs petits services spécialisés.

## Résumé

La branche de similarité représente mes fondations communes.

Ces compétences sont importantes parce qu’elles reviennent dans plusieurs domaines :

- développement logiciel ;
- DevOps ;
- intelligence artificielle ;
- full stack ;
- cloud ;
- automatisation ;
- qualité logicielle.

## Formule

Certificats différents → cours similaires → compétences communes → fondations professionnelles
