## 5. Formule de progression

Je ne mémorise pas seulement les cours.

Je transforme chaque cours en :

1. compétence ;
2. explication simple ;
3. modèle de code ;
4. mini-projet ;
5. preuve professionnelle.

---

## 6. Objectif professionnel

Engineer Method OS doit devenir :

- mon carnet de compétences ;
- mon carnet de modèles de code ;
- mon système d’organisation ;
- mon portfolio ;
- une preuve que je peux relier plusieurs cours dans un projet professionnel.
