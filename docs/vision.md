# Engineer Method OS
Engineer Method OS est une application conçue pour transformer mes cours, certificats, concepts, modèles de code et projets en une méthode professionnelle d'apprentissage et de travail.L'objectif n'est pas seulement de mémoriser du code, mais de comprendre, organiser, appliquer et prouver mes compétences.
## Objectif principal
Transformer chaque cours en compétence, chaque compétence en modèle,chaque modèle en projet, et chaque projet en preuve professionnelle.

## Méthode

Engineer Method OS repose sur deux branches principales :

1.Branche de similarité
-Elle regroupe les cours communs entre les certificats.
-Elle montre les fondations que je dois maîtriser.

2.Branche de différenciation 
-Elle montre ce que chaque certificat apporte de spécial.
-Elle permet de comprendre les spécialités: Full Stack, JavaScript, DevOps, IA, Cloud, Testing, etc.

## Structure de pensée

Certificat → Cours → Concept → Explication → Modèle de code → Projet → Preuve professionnelle

## Vision 

Créer un système qui m'aide à devenir organisé, plus professionnel et capable d'appliquer mes connaissances dans différents domaines.
