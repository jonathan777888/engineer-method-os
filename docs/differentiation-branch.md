# Branche de différenciation

La branche de différenciation montre ce que chaque certificat apporte de spécial.

Son objectif est de répondre à la question :

> Qu’est-ce que chaque certificat m’apporte de différent ?

Même si plusieurs certificats partagent des cours similaires, chaque certificat a une identité particulière.

---

## 1. IBM Full-Stack JavaScript Developer

### Spécialité

Développement web complet avec JavaScript.

### Compétences principales

- JavaScript
- React
- Node.js
- Express
- MongoDB
- API
- Applications full-stack
- Projet capstone JavaScript

### Explication simple

Ce certificat m’aide à construire une application web complète avec JavaScript, du frontend jusqu’au backend.

### Identité professionnelle

Développeur web full-stack JavaScript junior.

---

## 2. IBM DevOps and Software Engineering

### Spécialité

Organisation professionnelle, automatisation, tests, sécurité et déploiement.

### Compétences principales

- DevOps
- Agile et Scrum
- Linux
- Shell scripting
- GitHub
- Tests
- CI/CD
- Docker
- Kubernetes
- OpenShift
- Sécurité applicative
- Monitoring et observability

### Explication simple

Ce certificat m’aide à travailler comme un professionnel capable d’automatiser, tester, sécuriser et déployer une application.

### Identité professionnelle

DevOps junior, QA junior, cloud support ou automation assistant.

---

## 3. IBM AI Developer

### Spécialité

Développement d’applications avec intelligence artificielle.

### Compétences principales

- Intelligence artificielle
- IA générative
- Prompt engineering
- Python
- Flask
- Applications IA
- Chatbots
- Outils intelligents

### Explication simple

Ce certificat m’aide à comprendre et intégrer l’intelligence artificielle dans des applications.

### Identité professionnelle

Développeur IA junior ou assistant en automatisation IA.

---

## 4. IBM Full Stack Software Developer

### Spécialité

Développement logiciel complet avec frontend, backend, base de données, cloud et déploiement.

### Compétences principales

- HTML
- CSS
- JavaScript
- React
- Node.js
- Python
- Flask
- Django
- SQL
- NoSQL
- GitHub
- Docker
- Kubernetes
- Microservices
- Serverless
- Sécurité
- SaaS
- Capstone full-stack

### Explication simple

Ce certificat est le plus général. Il m’aide à comprendre tout le cycle de création d’une application moderne.

### Identité professionnelle

Développeur logiciel full-stack junior.

---

## Résumé de la branche

Chaque certificat ajoute une spécialité :

- Full-Stack JavaScript Developer → spécialité JavaScript
- DevOps and Software Engineering → spécialité automatisation, déploiement et qualité
- AI Developer → spécialité intelligence artificielle
- Full Stack Software Developer → spécialité développement logiciel complet

## Formule

Certificats différents → spécialités différentes → identité professionnelle plus forte
